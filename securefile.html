🔐 Secure File Encryption & Decryption Tool

This web-based application allows you to securely encrypt and decrypt files directly in your browser — no installation or upload required.

🔧 How to Use:

  • 📁 Upload a File: Drag and drop or select a file you want to protect.
  • 🔑 Enter a Password: Choose a strong password — this will be used as your encryption key.
  • 🛡️ Encrypt: Click the Encrypt button to convert your file into a secure, unreadable format.
  • 🔓 Decrypt: To access the original file again, re-upload the encrypted file, enter the correct password, and click Decrypt.
  • ⏰ Optional Expiration: Set a file access expiration time for temporary access (for display purposes only).
  • 📊 Progress Tracking: Watch the progress bar for feedback during the encryption/decryption process.
  • 📷 QR Code Sharing: Generate a QR code to share the encrypted file link securely (if sharing is enabled).

✅ 100% Private: All encryption and decryption is done locally in your browser. No files or passwords are transmitted or stored anywhere online.

Tip: Use a unique, strong password and store it safely — losing it means losing access to the encrypted file.
